# My Calendar Viewer

GitHub Pagesで動作するシンプルなカレンダービューアー。

## 機能

- 🔗 **Google Calendar公開URL対応** - CORSプロキシ経由でicalデータを取得
- 📋 **データ貼り付け** - icalデータを直接入力
- 📁 **ファイルアップロード** - .icsファイルを読み込み
- 📅 **複数表示モード** - 月/週/日表示切替
- 🔒 **セキュリティ** - マスク機能、データはブラウザ内のみで処理

## 使い方

### 1. Google Calendar公開URL方式（推奨）

1. Google Calendarで公開設定
   - カレンダー設定 → 「アクセス権の設定」→「一般に公開する」
   - 「公開用アドレス」のicalリンクをコピー
   
2. アプリケーションでURL入力
   - 「🔗 URLから読み込み」タブを選択
   - ical URLを貼り付けて「カレンダーを表示」

### 2. データ貼り付け方式

- icalデータをテキストエリアに直接貼り付け

### 3. ファイルアップロード方式

- .icsファイルを選択してアップロード

## 技術スタック

- FullCalendar v6
- ical.js
- Tailwind CSS
- CORS Proxy (allorigins.win, corsproxy.io)

## 注意事項

- **公開カレンダーのみ対応** - 秘密アドレスは使用しないでください
- CORSプロキシは無料サービスのため、安定性は保証されません
- データはブラウザのメモリ上のみで処理され、サーバーには送信されません
